# Linear Shortcuts

> **Note:** Linear shortcuts work in the Linear web app and desktop app.

## Navigation

| Shortcut | Action |
| --- | --- |
| G then I | Go to Inbox |
| G then M | Go to My Issues |
| G then T | Go to My Drafts |
| G then V | Go to Views |
| G then P | Go to Projects |
| G then B | Go to Roadmaps |
| G then C | Go to Cycles |
| G then E | Go to Members |
| G then S | Go to Settings |
| G then A | Go to All Issues |
| G then H | Go to Team Issues |
| G then D | Go to Dashboard |
| G then R | Go to Backlog |
| Ctrl + K | Open command palette |
| Ctrl + / | Open search |

## Issue Actions

| Shortcut | Action |
| --- | --- |
| C | Create new issue |
| Ctrl + Enter | Submit / save issue |
| E | Edit issue |
| X | Select issue |
| Backspace | Delete selected issue(s) |
| Ctrl + C | Copy issue URL |
| Ctrl + Shift + C | Copy issue ID |
| Ctrl + D | Duplicate issue |
| Ctrl + Z | Undo last action |
| Shift + I | Set issue priority to Urgent |
| Shift + O | Set issue priority to No Priority |
| 1 | Set issue priority to Urgent |
| 2 | Set issue priority to High |
| 3 | Set issue priority to Medium |
| 4 | Set issue priority to Low |
| 0 | Set issue priority to No priority |
| A | Assign to me |
| S | Change issue status |
| L | Add label |
| P | Set project |
| D | Set due date |
| Ctrl + M | Move to another team |
| Shift + M | Set milestone |
| Shift + E | Estimate issue |
| F | Add to favorites |
| Shift + S | Subscribe to issue |
| T | Set issue type |
