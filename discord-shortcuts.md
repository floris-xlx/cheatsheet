# Discord Shortcuts

> **Note:** Shortcuts listed are for Windows/Linux. For macOS, replace `Ctrl` with `Cmd` and `Alt` with `Option`.

## Navigation

| Shortcut | Action |
| --- | --- |
| Ctrl + K | Quick switcher (jump to server/channel) |
| Ctrl + Tab | Move to next unread channel |
| Ctrl + Shift + Tab | Move to previous unread channel |
| Alt + Up | Navigate to previous channel/DM |
| Alt + Down | Navigate to next channel/DM |
| Alt + Shift + Up | Navigate to previous unread channel |
| Alt + Shift + Down | Navigate to next unread channel |
| Ctrl + Alt + Up | Navigate to previous server |
| Ctrl + Alt + Down | Navigate to next server |
| Ctrl + 1 through Ctrl + 9 | Go to server (1–9) on sidebar |
| Ctrl + Shift + M | Toggle mute |
| Ctrl + Shift + D | Toggle deafen |
| Ctrl + Shift + H | Toggle Discord home (friends/DMs) |
| Escape | Mark channel as read |
| Shift + Escape | Mark server as read |

## Messaging

| Shortcut | Action |
| --- | --- |
| Up | Edit last message |
| Shift + Enter | Start new line in message |
| @ | Mention someone |
| # | Link a channel |
| : | Use emoji shortcode |
| Ctrl + Enter | Send message (when enabled) |
| Ctrl + F | Search in current channel |
| Ctrl + R | React to last message |
| Ctrl + E | Open emoji picker |
| Ctrl + Shift + E | Open emoji picker |
| Ctrl + U | Upload a file |
| Ctrl + P | Pin a message (in message context menu) |
| /spoiler | Send a spoiler message |

## Search

| Shortcut | Action |
| --- | --- |
| Ctrl + F | Open search in current server/channel |
| Ctrl + K | Open quick switcher search |
| from: | Filter by user in search |
| in: | Filter by channel in search |
| has:link | Filter messages with links |
| has:embed | Filter messages with embeds |
| has:file | Filter messages with files |
| has:image | Filter messages with images |
| has:video | Filter messages with video |
| has:sound | Filter messages with audio |
| has:sticker | Filter messages with stickers |
| before: | Filter messages before date |
| during: | Filter messages on date |
| after: | Filter messages after date |
| pinned:true | Filter pinned messages |
| mention:@me | Filter messages that mention you |
